# GPTHub — Быстрый запуск

## Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop) (macOS / Windows / Linux)
- MWS API ключ

## Запуск — 3 шага

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd open-webui
```

### 2. Запустить одной командой

```bash
bash start.sh
```

Скрипт сам:

- проверит Docker
- спросит MWS API ключ (один раз)
- скачает или соберёт образ
- запустит приложение

### 3. Открыть в браузере

```
http://localhost:3000
```

Зарегистрируйтесь — первый пользователь автоматически получает права администратора.

## Что можно попробовать

| Функция                   | Как                                                                                   |
| ------------------------- | ------------------------------------------------------------------------------------- |
| **Auto Mode**             | Выберите модель «Auto Router» — система сама выберет лучшую модель для вашего запроса |
| **Чат с текстом**         | Любой текстовый запрос                                                                |
| **Генерация изображений** | «Нарисуй...» или выберите «Image Generator»                                           |

| **RAG (документы)**       | Прикрепите файл кнопкой скрепки                                                       |

## Остановить

```bash
docker compose -f docker-compose.judge.yaml down
```

## Удалить данные полностью

```bash
docker compose -f docker-compose.judge.yaml down -v
```

---

## Если что-то пошло не так

**Порт 3000 занят:**

```bash
# Изменить порт в docker-compose.judge.yaml:
# ports: - "3001:8080"
# затем открыть http://localhost:3001
```

**Проблемы с API:**

- Убедитесь что MWS API ключ действителен
- Ключ хранится в файле `.env.judge` (можно отредактировать вручную)
- После изменения ключа перезапустите: `bash start.sh`

**Логи:**

```bash
docker compose -f docker-compose.judge.yaml logs -f
```
